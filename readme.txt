Simple facebook comment plugin.

== Description ==
Simple facebook comment plugin is easy to use on your post, page, or text widget content

== Docs & Support ==

Just enter facebook api key and enable it
copy paste shortcode to any page


== Installation ==

1. Upload the entire `comment-kb` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.

You will find 'Facebook Comment' menu in your WordPress admin panel.
